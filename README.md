# Projeto de Composição Web
## O que foi feito?
Foi desenvolvida uma aplicação centrada na gestão de filmes, com diferentes funcionalidades associadas ao tipo de utilizador autenticado. 
O projeto incluiu um sistema de autenticação com dois perfis distintos, permitindo a definição de permissões diferentes consoante o papel de cada utilizador.

Um dos perfis tinha permissões de administração, podendo criar, editar e eliminar filmes da plataforma. O outro perfil estava direcionado para a consulta do catálogo, 
permitindo visualizar os filmes disponíveis e selecionar os seus favoritos, o que tornou possível distinguir de forma prática as operações de gestão das operações de utilização comum.

Além disso, a aplicação integrava um gerador de PDF, utilizado para exportar informação de forma estruturada. Este projeto permitiu-me consolidar conhecimentos na área de Composição Web,
sobretudo ao nível da autenticação, controlo de acessos por perfis e organização funcional de uma aplicação com diferentes tipos de interação.

## Quem participou?
Este projeto foi realizado por mim no 1º ano da Lincenciatura de Informática Web, Móvel e na Nuvem.

## Que linguagens de programação foram utilizadas?

- PHP
- CSS
- SQL
- FPDF

## Como inicializar o projeto?
Com o auxilio do XAMPP inicia-se o servidor web com o ficheiro "home.php".
